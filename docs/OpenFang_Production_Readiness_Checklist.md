# Production Readiness Checklist (ACME MVP)

- Verify all OpenFang adoption components are integrated into the ACME MVP branch.
- Validate governance artifacts are signed off and in final form.
- Validate Hands manifest, runner, and API surface are stable and tested.
- Validate MCP bootstrap, memory wiring, and embedding wiring are complete.
- Validate security primitives are wired and tests exist for HMAC, audit trail, prompt scanning, gating, and VM sandbox integration.
- Validate UI dashboards and security panels render correctly in the UI.
- Validate migration onboarding and rollout plans with pilot participants.
- Validate performance metrics and alerting are wired to the adoption dashboards.
- Sign-off from the Steering Committee.
