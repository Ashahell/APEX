# OpenFang Rollout Plan (ACME MVP)

- Objective: stage the production rollout of the OpenFang adoption for ACME MVP with phased cohorts and concrete rollback.
- Scope: Hands MVP, MCP bootstrap, memory scaffolding, and the API surface. Rollout gating via feature flags.
- Rollout Phases:
- Phase 1: Pilot (2 teams) — MVP surface; monitor security signals and adoption metrics
- Phase 2: Expanded rollout to additional teams — integrate feedback and adjust governance crosswalks
- Phase 3: General availability — stable adoption, evergreen governance
- Rollback plan: if critical issues appear, revert changes and roll back to the last stable release with runbooks
- Metrics and success criteria: adoption rate, stability, performance, audit-trail integrity
