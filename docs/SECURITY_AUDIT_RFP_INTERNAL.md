Security Audit RFP - Internal (Phase 1)
=====================================
Overview
- Internal prompt and requirements for requesting external security audit services.

Scope
- Similar to SOW, but formatted for internal procurement: scope, required deliverables, milestones, data sharing, NDA, acceptance criteria, and evaluation process.

Submission Requirements
- Company info, references, proposed team, timeline, cost, NDA terms.
- Evidence of prior audits, certifications, and security practices.

Evaluation & Selection
- Use SECURITY_AUDIT_EVAL_CRITERIA.md to score proposals.
- Document rationale for selection decisions.

Next Steps
- Once vendor selected, finalize SOW and NDA using templates in this folder.
