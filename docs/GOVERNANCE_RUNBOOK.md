Governance Runbook
================

- Cadence: quarterly governance reviews, annual security audits, and ad-hoc incident response governance.
- Roles: governance board, security lead, project manager, technical leads.
- Artifacts: governance charter, cadence doc, crosswalks, audit trails.
- Decision process: majority vote or delegated authority for critical security changes.
- Output: sign-off on major milestones, updated risk register, and runbooks.
